# SkinRoutine

A personal skincare diary: track products, log AM/PM routines, flag ingredients, and get suggestions based on your skin type and concerns.

## Features

- Google sign-in with optional cloud sync (Supabase)
- Onboarding quiz for skin profile and starter products
- Product library with ingredient flags, PAO/expiry tracking, and conflict warnings
- Routine log with skin ratings and a 7-day trend chart
- Personalised product recommendations and UK retailer links

## Scripts

Data is stored locally in the browser (IndexedDB) and synced to Supabase when signed in.
